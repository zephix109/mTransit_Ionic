# mTransit_Ionic
<b>Mini capstone project for Soen 390</b>

###For running locally
1 - ensure u have 4 things installed 
    a) npm -v
    b) node -v
    c) cordova -v
    d) ionic -v

2 - Include these plugins
    a) ionic plugin add cordova-plugin-network-information
    b) ionic plugin add cordova-plugin-geolocation
